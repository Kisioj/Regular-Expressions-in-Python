Regular expressions in Python

Regular expressions concept

  • not strictly related to Python or any other programming language
  • came into common use with Unix text-processing tools

What are regular expressions (REs, regexes, regexps)?

  • sequences of characters that define search patterns
    • some characters have special meaning (e.g.: "." means any character)
  • usually used for:
    • searching in text
    • transforming text
    • validating text data

Where are regular expressions used?

  • text editors
  • other tools (grep, sed, etc.)
  • programming

Regular expressions in Python

Regular expressions (called REs, or regexes, or regex patterns) are essentially a tiny, highly specialized programming language embedded inside Python and made available through the re module.

re module's functions

search(pattern, string, flags=0)
    Search a string for the presence of a pattern.

fullmatch(pattern, string, flags=0)
    Match a regular expression pattern to all of a string.

findall(pattern, string, flags=0)
    Find all occurrences of a pattern in a string.

Example

>>> html = '''<html><body>
...     <h1>Tytuł</h1>
...     <h1>Błąd</p>
...     <p>Par1</p><p>Par2</p>
... </body></html>'''
>>> re.findall(r'<h1>.*</h1>', html)
['<h1>Tytuł</h1>']
>>> re.findall(r'<h1>(.*)</h1>', html)
[('Tytuł', '</h1>')]
>>> re.findall(r'<h1>(.*)(</h1>)', html)
['Tytuł']

Example

>>> html = '''<html><body>
...     <h1>Tytuł</h1>
...     <h1>Błąd</p>
...     <p>Par1</p><p>Par2</p>
... </body></html>'''
>>> re.findall(r'<p>.*</p>', html)
['<p>Par1</p><p>Par2</p>']
>>> re.findall(r'<p>[^<]*</p>', html)
['<p>Par1</p>', '<p>Par2</p>']
>>> re.findall(r'<p>.*?</p>', html)
['<p>Par1</p>', '<p>Par2</p>']
['Par1', 'Par2']
>>> re.findall(r'<p>(.*?)</p>', html)

Example

>>> html = '''<html><body>
...     <h1>Tytuł</h1>
...     <h1>Błąd</p>
...     <p>Par1</p><p>Par2</p>
... </body></html>'''
>>> re.findall(r'<(p|h1)>(.*?)</(p|h1)>', html)
[('h1', 'Tytuł', 'h1'), ('h1', 'Błąd', 'p'), ('p', 'Par1', 'p'), ('p', 'Par2', 'p')]
>>> re.findall(r'<(p|h1)>(.*?)</\1>', html)
[('h1', 'Tytuł'), ('p', 'Par1'), ('p', 'Par2')]

re module's functions

search(pattern, string, flags=0)
    Search a string for the presence of a pattern.

fullmatch(pattern, string, flags=0)
    Match a regular expression pattern to all of a string.

findall(pattern, string, flags=0)
    Find all occurrences of a pattern in a string.

compile(pattern, flags=0)
    Compile a pattern into a RegexObject.

Example

>>> text = "Nie wiesz może gdzie znajdę Alę?"
>>> myregex = re.compile(r"Al[aięąu]")
>>> re.findall(myregex, text)
['Alę']
>>> from timeit import timeit
>>> timeit('myregex.findall(text)', globals=globals())
>>> myregex.findall(text)
['Alę']
0.5122056359495131
>>> timeit('re.findall(myregex, text)', globals=globals())
1.473552673556437
>>> timeit('re.findall(r"Al[aięąu]", text)', globals=globals())
0.9848888915498364
